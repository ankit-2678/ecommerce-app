🛒 E-Commerce App

A modern and responsive e-commerce web application built using React + Vite, featuring product browsing, filtering, and sorting functionalities.

🚀 Features

🏠 Clean and responsive Homepage UI
📦 Product Collection Page
🔍 Filter products by Category and Subcategory
↕️ Sort products by price (Low → High / High → Low)
⚡ Fast performance using Vite

🛠 Tech Stack

Frontend: React.js
Build Tool: Vite
Styling: Tailwind CSS
Language: JavaScript (ES6+)

📂 Project Structure

src/
 ├── assets/        # Images and static assets
 ├── components/    # Reusable UI components
 ├── context/       # Global state (ShopContext)
 ├── pages/         # Application pages (Home, Collection)
 ├── App.jsx        # Main app component
 └── main.jsx       # Entry point
