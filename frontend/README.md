# 🛒 E-Commerce Web App

A modern and responsive **E-Commerce Web Application** built using React. This project allows users to browse products, view detailed product pages, and explore a clean and intuitive user interface.

The project currently focuses on **frontend development**, with plans to integrate backend services and full shopping functionality.

---

## ✨ Features

* 🏠 Home page with product listings
* 🔍 Search functionality
* 🛍️ Product details page (size selection, related products)
* 📂 Collection page with filtering options
* ℹ️ About page
* 📞 Contact page
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Language:** JavaScript (ES6+)
* **Styling:** CSS / Tailwind CSS
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/ankit-2678/ecommerce-app.git
```

2. Navigate to frontend folder

```
cd frontend
```

3. Install dependencies

```
npm install
```

4. Run the application

```
npm run dev
```
